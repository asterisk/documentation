# SPEECH ACTIVATE GRAMMAR

### Synopsis

Activates a grammar.

### Description

Activates the specified grammar on the speech object.<br>


### Syntax


```

SPEECH ACTIVATE GRAMMAR GRAMMAR NAME 
```
##### Arguments


* `grammar name`
