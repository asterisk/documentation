# VERBOSE

### Synopsis

Logs a message to the asterisk verbose log.

### Description

Sends _message_ to the console via verbose message system. _level_ is the verbose level (1-4). Always returns '1'<br>


### Syntax


```

VERBOSE MESSAGE LEVEL 
```
##### Arguments


* `message`

* `level`
