# ODBCFinish()

### Synopsis

Clear the resultset of a sucessful multirow query.

### Description

For queries which are marked as mode=multirow, this will clear any remaining rows of the specified resultset.<br>


### Syntax


```

ODBCFinish(result-id)
```
##### Arguments


* `result-id`
