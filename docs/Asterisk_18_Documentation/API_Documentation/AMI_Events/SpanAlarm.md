# SpanAlarm

### Synopsis

Raised when an alarm is set on a DAHDI span.

### Syntax


```


Event: SpanAlarm
Span: <value>
Alarm: <value>

```
##### Arguments


* `Span` - The span on which the alarm occurred.<br>

* `Alarm` - A textual description of the alarm that occurred.<br>
