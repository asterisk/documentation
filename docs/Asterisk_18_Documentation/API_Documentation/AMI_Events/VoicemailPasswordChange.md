# VoicemailPasswordChange

### Synopsis

Raised in response to a mailbox password change.

### Syntax


```


Event: VoicemailPasswordChange
Context: <value>
Mailbox: <value>
NewPassword: <value>

```
##### Arguments


* `Context` - Mailbox context.<br>

* `Mailbox` - Mailbox name.<br>

* `NewPassword` - New password for mailbox.<br>
