# CoreShowChannelMapComplete

### Synopsis

Raised at the end of the CoreShowChannelMap list produced by the CoreShowChannelMap command.

### Syntax


```


Event: CoreShowChannelMapComplete
EventList: <value>
ListItems: <value>

```
##### Arguments


* `EventList` - Conveys the status of the command response list<br>

* `ListItems` - The total number of list items produced<br>
