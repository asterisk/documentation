# ListCommands

### Synopsis

List available manager commands.

### Description

Returns the action name and synopsis for every action that is available to the user.<br>


### Syntax


```


Action: ListCommands
ActionID: <value>

```
##### Arguments


* `ActionID` - ActionID for this transaction. Will be returned.<br>
