# PJSIPQualify

### Synopsis

Qualify a chan_pjsip endpoint.

### Description

Qualify a chan\_pjsip endpoint.<br>


### Syntax


```


Action: PJSIPQualify
ActionID: <value>

```
##### Arguments


* `ActionID` - ActionID for this transaction. Will be returned.<br>

* `Endpoint` - The endpoint you want to qualify.<br>
