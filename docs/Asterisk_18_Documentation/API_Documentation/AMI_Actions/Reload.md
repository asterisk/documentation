# Reload

### Synopsis

Send a reload event.

### Description

Send a reload event.<br>


### Syntax


```


Action: Reload
ActionID: <value>
Module: <value>

```
##### Arguments


* `ActionID` - ActionID for this transaction. Will be returned.<br>

* `Module` - Name of the module to reload.<br>
