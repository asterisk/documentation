# QueueRemove

### Synopsis

Remove interface from queue.

### Description


### Syntax


```


Action: QueueRemove
ActionID: <value>
Queue: <value>
Interface: <value>

```
##### Arguments


* `ActionID` - ActionID for this transaction. Will be returned.<br>

* `Queue` - The name of the queue to take action on.<br>

* `Interface` - The interface (tech/name) to remove from queue.<br>
