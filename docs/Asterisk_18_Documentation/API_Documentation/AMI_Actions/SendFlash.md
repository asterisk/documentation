# SendFlash

### Synopsis

Send a hook flash on a specific channel.

### Description

Sends a hook flash on the specified channel.<br>


### Syntax


```


Action: SendFlash
ActionID: <value>
Channel: <value>
[Receive: <value>]

```
##### Arguments


* `ActionID` - ActionID for this transaction. Will be returned.<br>

* `Channel` - Channel name to send hook flash to.<br>

* `Receive` - Emulate receiving a hook flash on this channel instead of sending it out.<br>
