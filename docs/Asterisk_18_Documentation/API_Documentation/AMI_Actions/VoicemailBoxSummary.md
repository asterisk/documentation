# VoicemailBoxSummary

### Synopsis

Show the mailbox contents of given voicemail user.

### Description

Retrieves the contents of the given voicemail user's mailbox.<br>


### Syntax


```


Action: VoicemailBoxSummary
ActionID: <value>
Context: <value>
Mailbox: <value>

```
##### Arguments


* `ActionID` - ActionID for this transaction. Will be returned.<br>

* `Context` - The context you want to check.<br>

* `Mailbox` - The mailbox you want to check.<br>
