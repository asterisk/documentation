# DAHDIRestart

### Synopsis

Fully Restart DAHDI channels (terminates calls).

### Description

Equivalent to the CLI command "dahdi restart".<br>


### Syntax


```


Action: DAHDIRestart
ActionID: <value>

```
##### Arguments


* `ActionID` - ActionID for this transaction. Will be returned.<br>
