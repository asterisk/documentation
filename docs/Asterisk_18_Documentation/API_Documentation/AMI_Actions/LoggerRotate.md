# LoggerRotate

### Synopsis

Reload and rotate the Asterisk logger.

### Description

Reload and rotate the logger. Analogous to the CLI command 'logger rotate'.<br>


### Syntax


```


Action: LoggerRotate
ActionID: <value>

```
##### Arguments


* `ActionID` - ActionID for this transaction. Will be returned.<br>
