# CSV_QUOTE()

### Synopsis

Quotes a given string for use in a CSV file, escaping embedded quotes as necessary

### Description

Example: $\{CSV\_QUOTE("a,b" 123)\} will return """a,b"" 123"<br>


### Syntax


```

CSV_QUOTE(string)
```
##### Arguments


* `string`
