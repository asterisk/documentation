# GLOBAL_EXISTS()

### Synopsis

Check if a global variable exists or not.

### Description

Returns '1' if global variable exists or '0' otherwise.<br>


### Syntax


```

GLOBAL_EXISTS(varname)
```
##### Arguments


* `varname` - Global variable name<br>
