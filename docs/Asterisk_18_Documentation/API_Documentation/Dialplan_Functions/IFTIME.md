# IFTIME()

### Synopsis

Temporal Conditional.

### Description

Returns the data following '?' if true, else the data following ':'<br>


### Syntax


```

IFTIME(timespec?retvalue)
```
##### Arguments


* `timespec`

* `retvalue`

    * `true`

    * `false`
