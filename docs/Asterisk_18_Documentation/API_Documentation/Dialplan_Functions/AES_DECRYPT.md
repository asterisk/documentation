# AES_DECRYPT()

### Synopsis

Decrypt a string encoded in base64 with AES given a 16 character key.

### Description

Returns the plain text string.<br>


### Syntax


```

AES_DECRYPT(key,string)
```
##### Arguments


* `key` - AES Key<br>

* `string` - Input string.<br>
