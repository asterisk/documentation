# GLOBAL_DELETE()

### Synopsis

Deletes a specified global variable.

### Description

Delete the global variable specified in _varname_. Will succeed if the global variable exists or not.<br>


### Syntax


```

GLOBAL_DELETE(varname)
```
##### Arguments


* `varname` - Global variable name<br>
