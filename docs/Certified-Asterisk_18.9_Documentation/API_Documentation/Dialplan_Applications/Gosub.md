# Gosub()

### Synopsis

Jump to label, saving return address.

### Description

Jumps to the label specified, saving the return address.<br>


### Syntax


```

Gosub([[context,]extension,]priority[(arg1,[...][argN]]))
```
##### Arguments


* `context`

* `extension`

* `priority`

    * `arg1[,arg1...]` **required**

    * `argN`
