# TestClient()

### Synopsis

Execute Interface Test Client.

### Description

Executes test client with given _testid_. Results stored in */var/log/asterisk/testreports/<testid>-client.txt*<br>


### Syntax


```

TestClient(testid)
```
##### Arguments


* `testid` - An ID to identify this test.<br>
