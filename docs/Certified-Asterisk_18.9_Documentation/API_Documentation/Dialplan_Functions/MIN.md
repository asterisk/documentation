# MIN()

### Synopsis

Returns the minimum of two numbers.

### Description

Returns the minimum of two numbers _num1_ and _num2_.<br>

Example: Set(min=$\{MIN(7,4)\}); Sets the min variable equal to 4.<br>


### Syntax


```

MIN(num1,num2)
```
##### Arguments


* `num1`

* `num2`
