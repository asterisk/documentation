# ABS()

### Synopsis

Returns absolute value of a number.

### Description

Returns the absolute value of a number _num_.<br>

Example: Set(absval=$\{ABS(-13)\}); Sets the absval variable equal to 13.<br>


### Syntax


```

ABS(num)
```
##### Arguments


* `num`
