# CHANNEL_EXISTS()

### Synopsis

Checks if the specified channel exists.

### Description

Returns 1 if the channel _name\_or\_uid_ exists, 0 if not.<br>


### Syntax


```

CHANNEL_EXISTS(name_or_uid)
```
##### Arguments


* `name_or_uid` - The name or unique ID of the channel to check.<br>
