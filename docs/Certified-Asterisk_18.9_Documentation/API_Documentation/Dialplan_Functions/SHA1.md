# SHA1()

### Synopsis

Computes a SHA1 digest.

### Description

Generate a SHA1 digest via the SHA1 algorythm.<br>

Example: Set(sha1hash=$\{SHA1(junky)\})<br>

Sets the asterisk variable sha1hash to the string '60fa5675b9303eb62f99a9cd47f9f5837d18f9a0' which is known as his hash<br>


### Syntax


```

SHA1(data)
```
##### Arguments


* `data` - Input string<br>
