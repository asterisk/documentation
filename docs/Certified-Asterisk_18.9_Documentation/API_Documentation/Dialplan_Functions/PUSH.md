# PUSH()

### Synopsis

Appends one or more values to the end of a variable containing delimited text

### Description

Example: Set(PUSH(array)=one,two,three) would append one, two, and three to the end of the values stored in the variable "array".<br>


### Syntax


```

PUSH(varname[,delimiter])
```
##### Arguments


* `varname`

* `delimiter`
