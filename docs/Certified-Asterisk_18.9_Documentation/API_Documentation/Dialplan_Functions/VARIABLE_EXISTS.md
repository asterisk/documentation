# VARIABLE_EXISTS()

### Synopsis

Check if a dialplan variable exists or not.

### Description

Returns '1' if channel variable exists or '0' otherwise.<br>


### Syntax


```

VARIABLE_EXISTS(varname)
```
##### Arguments


* `varname` - Channel variable name<br>
