# IF()

### Synopsis

Check for an expresion.

### Description

Returns the data following '?' if true, else the data following ':'<br>


### Syntax


```

IF(expresion?retvalue)
```
##### Arguments


* `expresion`

* `retvalue`

    * `true`

    * `false`
