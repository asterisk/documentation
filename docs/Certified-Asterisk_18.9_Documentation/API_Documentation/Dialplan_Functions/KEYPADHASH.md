# KEYPADHASH()

### Synopsis

Hash the letters in string into equivalent keypad numbers.

### Description

Example: $\{KEYPADHASH(Les)\} returns "537"<br>


### Syntax


```

KEYPADHASH(string)
```
##### Arguments


* `string`
