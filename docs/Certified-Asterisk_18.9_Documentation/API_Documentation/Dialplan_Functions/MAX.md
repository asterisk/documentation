# MAX()

### Synopsis

Returns the maximum of two numbers.

### Description

Returns the maximum of two numbers _num1_ and _num2_.<br>

Example: Set(max=$\{MAX(4,7)\}); Sets the max variable equal to 7.<br>


### Syntax


```

MAX(num1,num2)
```
##### Arguments


* `num1`

* `num2`
