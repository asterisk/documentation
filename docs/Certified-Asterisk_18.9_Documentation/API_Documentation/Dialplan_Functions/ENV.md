# ENV()

### Synopsis

Gets or sets the environment variable specified.

### Description

Variables starting with 'AST\_' are reserved to the system and may not be set.<br>


### Syntax


```

ENV(varname)
```
##### Arguments


* `varname` - Environment variable name<br>
