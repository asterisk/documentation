# DELETE()

### Synopsis

Deletes a specified channel variable.

### Description

Delete the channel variable specified in _varname_. Will succeed if the channel variable exists or not.<br>


### Syntax


```

DELETE(varname)
```
##### Arguments


* `varname` - Channel variable name<br>
