# VoicemailRemove

### Synopsis

Remove Voicemail from mailbox folder.

### Description

Remove a given Voicemail from a user's Mailbox Folder.<br>


### Syntax


```


Action: VoicemailRemove
ActionID: <value>
Context: <value>
Mailbox: <value>
Folder: <value>
ID: <value>

```
##### Arguments


* `ActionID` - ActionID for this transaction. Will be returned.<br>

* `Context` - The context of the Voicemail you want to remove.<br>

* `Mailbox` - The mailbox of the Voicemail you want to remove.<br>

* `Folder` - The Folder containing the Voicemail you want to remove.<br>

* `ID` - The ID of the Voicemail you want to remove.<br>
