# CoreShowChannelMap

### Synopsis

List all channels connected to the specified channel.

### Description

List all channels currently connected to the specified channel. This can be any channel, including Local channels, and Local channels will be followed through to their other half.<br>


### Syntax


```


Action: CoreShowChannelMap
Channel: <value>

```
##### Arguments


* `Channel` - The channel to get the mapping for. Requires a channel name.<br>
