# BlindTransfer

### Synopsis

Blind transfer channel(s) to the given destination

### Description

Redirect all channels currently bridged to the specified channel to the specified destination.<br>


### Syntax


```


Action: BlindTransfer
Channel: <value>
Context: <value>
Exten: <value>

```
##### Arguments


* `Channel`

* `Context`

* `Exten`
