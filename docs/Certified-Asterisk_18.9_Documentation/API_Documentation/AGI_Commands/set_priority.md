# SET PRIORITY

### Synopsis

Set channel dialplan priority.

### Description

Changes the priority for continuation upon exiting the application. The priority must be a valid priority or label.<br>


### Syntax


```

SET PRIORITY PRIORITY 
```
##### Arguments


* `priority`
