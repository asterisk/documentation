# ASYNCAGI BREAK

### Synopsis

Interrupts Async AGI

### Description

Interrupts expected flow of Async AGI commands and returns control to previous source (typically, the PBX dialplan).<br>


### Syntax


```

ASYNCAGI BREAK 
```
