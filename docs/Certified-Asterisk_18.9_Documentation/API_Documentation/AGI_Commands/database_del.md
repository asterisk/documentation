# DATABASE DEL

### Synopsis

Removes database key/value

### Description

Deletes an entry in the Asterisk database for a given _family_ and _key_.<br>

Returns '1' if successful, '0' otherwise.<br>


### Syntax


```

DATABASE DEL FAMILY KEY 
```
##### Arguments


* `family`

* `key`
