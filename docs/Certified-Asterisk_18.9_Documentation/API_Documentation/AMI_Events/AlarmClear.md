# AlarmClear

### Synopsis

Raised when an alarm is cleared on a DAHDI channel.

### Syntax


```


Event: AlarmClear
DAHDIChannel: <value>

```
##### Arguments


* `DAHDIChannel` - The DAHDI channel on which the alarm was cleared.<br>

    /// note
